# Redbag Showcase

This folder contains wiki-style feature images for the Redbag Bukkit plugin. They are meant to help server owners understand the plugin behavior, commands, GUI, and configurable rules.

## Images

- `01-overview.png`: Core redbag modes and claim flow.
- `02-item-gui.png`: Full speed-redbag item-selection GUI.
- `03-left-click-claim.png`: Claiming a speed redbag by left-clicking while holding the configured item.
- `04-configurable.png`: Configurable expiry, claim items, messages, item lists, and broadcasts.

The final images use a generated documentation layout. Raw captured screenshots are kept in `raw/` as source material for GUI/title references in future redesigns.
