# Redbag Showcase

This folder contains generated showcase images for the Redbag Bukkit plugin. They can be used in the project README, plugin listing pages, or server introduction posts.

## Images

- `01-overview.png`: Core redbag flow overview.
- `02-item-gui.png`: Normal redbag item-selection GUI.
- `03-left-click-claim.png`: Claiming a redbag by left-clicking while holding the configured item.
- `04-configurable.png`: Configurable expiry, claim items, messages, and broadcasts.

The images are Minecraft-style generated assets, so they can ship with the repository without depending on a live client screenshot session.
